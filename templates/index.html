Лексический анализатор

Распознавание логических выражений

Вариант 8

  • Идентификаторы (a, b, x, result, ...)
  • Константы: 0 и 1
  • Присваивание: :=
  • Операции: or, xor, and, not
  • Скобки: ( )
  • Разделитель: ;
  • Комментарии: // и /* ... */

Входной код

Инструкция:

Введите логические выражения и нажмите "Анализировать". Каждое выражение должно заканчиваться точкой с запятой (;). Поддерживаются комментарии вида // и /* */.
